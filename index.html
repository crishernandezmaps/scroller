Visualizacion en Scroller
Una visualizacion de este tipo permite que el usuario en vertical navegue de forma intuitiva la informacion.
A medida que se avanza el usuario navegando hacia abajo, va descubriendo las diversas graficas.
La narrativa contiene una estructura lineal que es facil de digerir.
Se puede jugar con las transiciones.
Se pueden incluir diversos tipos de graficas.
